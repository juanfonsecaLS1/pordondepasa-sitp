# ¿Por Dónde Pasa? - SITP

Interactive web application to explore bus routes in Bogotá's public transport system (SITP). Search routes by stop codes, click on the map to see nearby routes, and view frequency information.

**Live site:** https://juanfonsecaLS1.github.io/pordondepasa-sitp/

## Local Development

```bash
cd gtfs-app
npm install
npm run dev
```

## Project Structure

- `gtfs-app/` - React + TypeScript + Vite web application
- `pre-processing/` - Python scripts for GTFS data processing
